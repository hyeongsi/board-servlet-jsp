# 📁 게시판 프로젝트 (servlet)
> Servlet/Jsp를 이용한 게시판 프로젝트 입니다.
<br/>

## 🎯 개요
> * Java 웹 개발의 기반 기술인 Servlet을 동작 방식 학습을 위해 프로젝트를 진행.<br><br>
> * Servlet 기반 프로젝트와 Spring MVC 프로젝트의 동작 방식 차이를 비교하여<br>
> Servlet 대신 Spring MVC를 쓰는 이유를 찾기 위함.<br>
<br/>
